# Fallout 4 Achievement Tracker

A simple browser-based tracker for Fallout 4 achievements. Check off achievements as you earn them, with progress saved locally in your browser.

## Features

- Lists all Fallout 4 achievements with descriptions and images
- Click an achievement to toggle it as completed
- Progress is persisted via `localStorage`
- Dark/light theme support via [Pico CSS](https://picocss.com/)

## Usage

Open `index.html` in a browser — no build step or server required.

## Project Structure

```
index.html          # Main app
achievements.json   # Achievement data (name, description, image path)
images/             # Achievement artwork
```
